# focusapp
AI-powered focus space that uses real-time face detection to keep you productive
**Real-Time AI Face Detection** to ensure you stay focused on your tasks. If you look away or leave your desk, the app will alert you!

## 🎀 Key Features
* **AI-Powered Monitoring:** Uses `Face-API.js` to detect your presence. If your face leaves the frame for more than 2 seconds, the buzzer sounds!
* **Personalized Experience:** Greeting and alerts dynamically adapt to the name you input.
* **Curated Affirmations:** Start every session with a random shot of motivation to boost your mood.
* **Elegant Visuals:** A carefully crafted palette of *Dusty Pink, Lilac, Beige,* and *Dark Choco* for a premium, calming user experience.
* **Session Tracking:** Simple timer to keep track of how long you've stayed in the "Zone."

## 🛠️ Technical Stack
* **Core:** HTML5 & JavaScript (ES6+)
* **Styling:** [Tailwind CSS](https://tailwindcss.com/)
* **AI Engine:** [Face-API.js](https://github.com/vladmandic/face-api) (Tiny Face Detector)
* **Typography:** Playfair Display & Quicksand via Google Fonts

## 🚀 Getting Started
1.  **Clone the Repository:**
    ```bash
    git clone [https://github.com/your-username/aesthetic-focus-space.git](https://github.com/your-username/aesthetic-focus-space.git)
    ```
2.  **Add Your Alert Sound:**
    Ensure your alarm file (`e2901f87-8953-4f43-8242-c8f067947246.mp3`) is in the same directory as `index.html`.
3.  **Launch:**
    Open `index.html` in any modern web browser (Chrome or Edge recommended for best camera performance).
4.  **Permissions:**
    Grant camera access when prompted to enable the AI detection features.
